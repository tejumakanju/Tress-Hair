# Tressé Hair

Luxury wig e-commerce storefront (Next.js) for Tressé Hair — Nigeria-first shipping and Flutterwave checkout.

## Getting started

```bash
npm install
cp .env.example .env.local
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

Fill in `.env.local` with Supabase and Flutterwave keys (see `.env.example`).
